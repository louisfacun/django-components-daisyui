# django-components-daisyui

Reusable DaisyUI v5 components for Django using django-components.

## Installation

```bash
pip install django-components-daisyui
```

## Requirements

- Python >= 3.10
- Django >= 5.0
- django-components >= 0.25


## Development

To build the package:

```bash
uv pip install build
uv run python -m build
